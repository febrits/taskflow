# TaskFlow — Kanban Project Management

A fully functional Kanban-style project management app with drag-and-drop, priorities, and custom columns.

**Live:** https://taskflow-livid-mu.vercel.app

## Features
- Drag-and-drop tasks between columns
- Custom columns (To Do, In Progress, Review, Done, etc.)
- Priority levels (Low, Medium, High, Urgent)
- Task descriptions, due dates, and labels
- Dark cyberpunk theme with glass morphism

## Tech Stack
React 19 · TypeScript · Tailwind CSS v4 · Vite · Framer Motion

## Getting Started
```bash
npm install
npm run dev
```
